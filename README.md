# Guild Navigators

A professional services website for AI implementation, strategic deconfusion, and research services.

## Project Structure

```
guild-navigators/
├── index.html # Main landing page
├── styles.css # Stylesheet
```

## Local Development

Simply open `index.html` in your browser to view the site.

## Deployment Options

This is a static site and can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
